FlexiSpace

FlexiSpace is a modern real-time space rental platform designed for discovering, booking, and managing flexible spaces such as parking spots, event halls, meeting rooms, workspaces, studios, and public utility areas.

Features

Rental Side

- User authentication
- Location-based property discovery
- Google Maps integration
- Real-time booking system
- Time-slot conflict prevention
- QR-based booking access
- Booking analytics dashboard
- Spending analysis

Provider Side

- Property management dashboard
- Add and manage spaces
- Upload property details
- Performance analytics
- Revenue tracking
- Occupancy insights
- Booking monitoring

Tech Stack

Frontend

- Next.js
- TypeScript
- Tailwind CSS
- Shadcn UI
- Framer Motion
- Recharts

Backend

- FastAPI
- SQLAlchemy
- SQLite/PostgreSQL
- JWT Authentication

Future Scope

- AI pricing optimization
- Smart occupancy prediction
- Dynamic recommendations
- Blockchain-based booking verification
- IoT smart access integration

Setup

Backend

cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload

Frontend

cd frontend
npm install
npm run dev
